# Data Professional Survey Dashboard (Power BI)

## Project Overview

This project analyzes survey responses from data professionals using Power BI. The dashboard provides insights into salaries, programming language preferences, career satisfaction, and challenges faced by professionals working in data-related roles.

## Dataset

The dataset contains survey responses from data professionals, including:

- Country
- Job Title
- Salary
- Age
- Favorite Programming Language
- Work/Life Balance Satisfaction
- Salary Satisfaction
- Difficulty Breaking Into Data

## Tools Used

- Power BI
- Power Query
- Data Modeling
- DAX
- Data Visualization

## Dashboard Features

### Country Analysis
Visualizes the distribution of survey participants by country.

### Salary Analysis
Compares average salaries across different data-related job titles.

### Programming Language Preferences
Shows the most popular programming languages among survey respondents.

### Career Entry Difficulty
Analyzes how difficult respondents found it to enter the data industry.

### Satisfaction Metrics
Measures:
- Happiness with Work/Life Balance
- Happiness with Salary

### KPI Cards
Displays:
- Total Survey Participants
- Average Age of Respondents

## Key Insights

- Data Scientists reported the highest average salaries.
- Python was the most popular programming language among respondents.
- Work/Life Balance satisfaction was higher than Salary satisfaction.
- Survey participants came from multiple countries, with the largest groups from the United States and India.
- Many respondents reported moderate to high difficulty entering the data field.
